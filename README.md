# 🌌 Gesture AR: The Future of Spatial Interaction

**Experience the absolute cutting edge of web-based AR.** Gesture AR is a high-performance, vanilla JavaScript spatial computing platform that transforms your physical environment into a digital playground using only your hands and a standard webcam.

---

## ✨ Features

- **☝️ Precision Drawing**: Crystal clear vector lines drawn via index finger tracking.
- **🌀 Holo Matrix**: A massive, high-fidelity 3D particle system reacting to hand tension and cursor position.
- **🧶 Dynamic Strings**: Procedural string art that vibrates and glows between your fingertips.
- **🪀 Realistic Rubbers**: Physical skipping-rope simulations with real-time sag and swirl physics.
- **🏝️ Liquid UI**: A sleek, Apple-inspired Dynamic Island interface for seamless mode switching.
- **🎨 Creative Control**: Integrated color pickers, glow sliders, and template selectors.
- **📸 Instant Export**: High-resolution canvas downloads to capture your creations.

## 🚀 Tech Stack

- **Three.js**: Powering the high-fidelity 3D particle system.
- **MediaPipe Hands**: Industry-leading ML for real-time skeletal hand tracking.
- **Web Audio API**: Procedural interactive soundscapes for physical feedback.
- **Vanilla JS & HTML5**: Zero-framework architecture for 60FPS hardware-bound performance.
